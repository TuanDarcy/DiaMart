# UI Components

## Nguyên tắc

- Component UI không chứa business logic quan trọng.
- Ưu tiên component nhỏ, rõ trách nhiệm.
- Không tạo component tái sử dụng khi mới chỉ dùng một lần và chưa có lý do rõ ràng.
- Component phải hỗ trợ responsive khi phù hợp.
